
# MEAN-Url-shortenner 

<h4> Using:</h4>
	- Mongodb 
	- Express
	- Angular
	- Node
	- ES6
	 
<h4>Install on Windows:</h4>
	- git clone https://github.com/AlexeyKagan/MEAN-stack-url-shortenner.git
	- cd mean-stack-url-shortenner
	- npm install
	- open mongodb server on port: 27017
	- type "gulp"
